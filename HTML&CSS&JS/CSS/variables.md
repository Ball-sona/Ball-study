# variables

css 내에서 color 값을 헥스 문자열로 관리하게 되면 코드가 많아졌을 때 관리하기 어려워질 수 있는데, 이런 경우 css 내부에서도 변수처럼 관리할 수 있다. 

만약 전역적으로 사용하고 싶다면,

```css
:root{
  --white:'#fff';
  --black:'#000';
  --primary:'#353535';
}

div{
  color:var(--white)
}
```

변수는 당연히 재사용 가능하다.
