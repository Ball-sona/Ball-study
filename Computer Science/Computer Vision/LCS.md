# LCS(Local Contour Sequence)

LCS는 영상의 지역적인 윤곽 또는 경계 정보를 나타내는 순서열을 의미한다. LCS는 영상의 픽셀 강도 변화를 나타내며, 영상 내 객체의 형태나 경계를 파악하는 데 사용된다.

- 객체 모양의 형태에 구애받지 않는다. 즉 볼록하고 오목한 윤곽을 가지는 제스처에 적합하다.
